Projects

IBM NestedLLAMA

The goal was to modify IBM's FMS Llama model to be compatible with Nested Tensors on forward inference to make them more memory efficient with variable length data on fine-tuning - a critical factor for deploying and training models on distributed and edge devices.

Performance was time and memory profiled. Additional tasks were to find bottlenecks in PyTorch's native implementation on NestedTensors. After completing the project we raised a PR on the sourcecode.

Technologies used: HuggingFace, IBM-FMS, PyTorch, CUDA, NVIDIA profiler.

This project focuses on real-time 3D pose estimation and correction on yoga positions. The goal is to develop a vectorized algorithm capable of accurately estimating and correcting yoga poses in real time, enabling users to receive immediate feedback on their posture during yoga practice.

The aim was to deploy a compute and time-intensive multi-thread UI application on a limited computing and limited power mobile edge device.
